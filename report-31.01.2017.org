Андрей Фильченков, руководитель исследовательской группы машинного обучения, доцент кафедры компьютерных технологий ИТМО, прочёл клубу лекцию о том, что такое Data Science и какие бывают разделы этой дисциплины.


Была представлена следующая иерархия дисциплин:


Data Science -> Data Analysis -> Machine Learning


Data Science – общее название дисциплины, включающее в себя анализ данных и работы с данными, такие как хранение и эффективный доступ к данным.

Data Science также включает в себя Data Engineering. Оба понятия не имеют устоявшихся определений. Data Engineering включает реализацию эффективного выполнения алгоритмов работы с данными.

Data Analysis состоит из:
- exploratory data analysis (разведочный анализ данных) – выделение важных переменных, внутренней структуры и т.п.
- confirmatory data analysis (проверка статистических гипотез)
- predictive data analysis (предсказательный анализ данных)
- data visualization

Разведочный анализ данных – выделение важных переменных, внутренней структуры данных и т.п. Тесно связано с визуализацией данных.

Проверка статистических гипотез и предсказательный анализ данных постоянно пересекаются по выполняемым задачам. Предсказательный анализ представляет собой множество методов для предсказания на основе исторических данных и используется в машинном обучении и data mining.

Анализ данных можно грубо охарактеризовать как статистическую обработку данных + машинное обучение.

Интеллектуальные системы – базы знаний, онтологии и прочее. Считается устаревшим.
